# Churn Tracker

Private iPhone app for tracking bank account churning offers with SwiftUI and SwiftData.

## Current v1 scaffold

- SwiftUI iPhone app
- SwiftData persistence
- Home dashboard focused on what to do next
- Offer CRUD
- Requirement tracking
- Offer timeline / event history
- Local analytics
- Local notification permission + deadline scheduling
- Screenshot attachment picker using `PhotosPicker`

## Notes

- Bundle identifier defaults to `com.braxtonmassengale.ChurnTracker`
- Set your Apple development team in Xcode before running on device
- The project is intentionally iPhone-only
- This is a strong v1 scaffold, not a finished App Store-grade product

## Next likely improvements

- Better notification preferences
- More granular requirement types
- Richer analytics similar to FlipTracker
- Import/export
- Better attachment gallery UX
